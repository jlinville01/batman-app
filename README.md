# Batman App

The objective of this application was to create a rails application to showcase various forms of testing including Front-End web automation, Back-End API automation, Unit/Fixture tests.

# Setup

1. Clone the repo.
2. `cd /batman-app`
3. `bundle install`
4. `rails s`
5. `rake db:create`
6. `rake db:migrate`
7. `rake db:seed`
